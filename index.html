Modul pelatihan di dalam analisis ini seluruhnya menggunakan Python di dalam notebook Google Colab, dengan pendekatan prompt-driven — di mana GenAI/LLM digunakan sebagai mitra pemrograman.

  • Dataset: TerraClimate — data iklim spatio-temporal global, dengan variabel seperti suhu, curah hujan, kelembaban, dan lainnya
  • Alat bantu: Google Colab + Python + GenAI (ChatGPT, Gemini, atau lainnya)
  • Satuan waktu: Bulanan dan Tahunan

Asumsi kunci: Tren iklim dapat mempengaruhi risiko banjir maupun panas kemarau secara signifikan, terutama melalui variabel suhu dan curah hujan

1 Overview

Pada akhir sesi ini, peserta akan mampu:

  1. Menggunakan Python di dalam Google Colab
  2. Membuka dan membaca data iklim ke dalam Google Colab
  3. Mengeksplorasi pola tren suhu dan curah hujan dengan grafik
  4. Memvisualisasikan pola suhu dan curah hujan secara spasial dengan peta
  5. Membuat peta bivariat untuk mengidentifikasi zona risiko iklim

2 Sebelum sesi dimulai

  1. Buka Colab notebook pada link di atas.
  2. Download dataset pada cell notebook yang tersedia.
  3. Pastikan Python dapat mengimpor numpy, pandas, geopandas, xarray, rioxarray, rasterio, matplotlib.
  4. Tinjau kembali alur tiap section di bawah sehingga semua prompt yang dijalankan sesuai.

3 Apa itu Google Colab?

Google Colab (singkatan dari “Colaboratory”) alat bantu gratis, lingkungan notebook berbasis browser dimana Anda dapat menjalankan kode Python secara online tanpa instalasi rumit di desktop/komputer lokal.

Colab dapat membantu dalam:

  1. Run notebook langsung dari link yang dibagikan
  2. Eksekusi kode dalam cell kecil dan inspeksi output langkah demi langkah
  3. Kombinasi teks naratif, prompt, kode, dan gambar dalam satu tempat
  4. Simpan salinan Anda sendiri di Google Drive untuk latihan berkelanjutan
TipTips untuk first-time users
  1. Gunakan Runtime > Run all untuk eksekusi semua cell secara berurutan
  2. Jika gagal, baca errornya, perbaiki prompt/kodenya, lalu jalankan ulang cell tersebut
  3. Jika runtime terputus, sambungkan kembali dan jalankan dari atas
